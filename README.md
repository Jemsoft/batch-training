Data buckets Blog Post


A recent addition to Monocular is the ability to create Data Buckets. Data Buckets provide the ability to store and organise your data and simplifies the process for training multiple detectors, face identifiers and other algorithms that are in development.

This tutorial provides a step by step guide on how to create and use data buckets when training an algorithm to detect stop signs in images using Monocular.
